How to get the code

The code is currently available in Remote-Learner's Github repositories. You can browse the code via our Github account here -- https://github.com/remotelearner

Direct access to each of the Remote-Learner ELIS Community repositories is available at the following URLs:

  • local_eliscore --- https://github.com/remotelearner/moodle-local_eliscore
  • local_elisprogram --- https://github.com/remotelearner/moodle-local_elisprogram
  • enrol_elis --- https://github.com/remotelearner/moodle-enrol_elis
  • block_elisadmin --- https://github.com/remotelearner/moodle-block_elisadmin
  • block_elisdashboard --- https://github.com/remotelearner/moodle-block_elisdashboard
  • local_elisreports --- https://github.com/remotelearner/moodle-local_elisreports

Optional:

  • local_datahub --- https://github.com/remotelearner/moodle-local_datahub
  • block_courserequest --- https://github.com/remotelearner/moodle-block_courserequest
  • block_enrolsurvey --- https://github.com/remotelearner/moodle-block_enrolsurvey
